# Amazon Product Web Scraping

This repository contains a Jupyter Notebook that demonstrates how to scrape product data from Amazon.in for PlayStation 5 search results. The project uses Python libraries such as BeautifulSoup, Requests, Pandas, and Selenium to extract product titles, prices, ratings, and links. The extracted data is saved to a CSV file for further analysis. The notebook includes detailed markdown explanations for each step, making it easy to follow and adapt for similar web scraping tasks.

## Features
- Scrapes Amazon search results for PlayStation 5 products
- Extracts product title, price, rating, and product link
- Handles dynamic content with Selenium for robust extraction
- Saves data to a CSV file
- Well-commented and explained for educational purposes

## Requirements
- Python
- Jupyter Notebook
- BeautifulSoup
- Requests
- Pandas
- Selenium
- webdriver-manager

## Usage
1. Clone the repository.
2. Install the required Python packages.
3. Open the notebook in Jupyter and run the cells.
4. The extracted data will be saved as `amazon_products.csv`.
